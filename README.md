ITA
=======
Integrated Toxicity Analysis System

Dependency
----------------
PostgresSQL 11
Python 3.0+
- bottle-pgsql
- bottle
- psycopg2
Marvin JS
JChem base



Start the server
----------------
Under the server folder, run

	python server.py
	open the url: http://localhost:8080/
Then your server should be up and running. Have fun!
